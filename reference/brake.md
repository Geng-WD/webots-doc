## Brake

Derived from `Device`.


```
Brake {
}
```

### Description

A `Brake` node can be used in a mechanical simulation in order to change the
friction of a joint. The `Brake` node can be inserted in the `device` field of a
`HingeJoint`, a `Hinge2Joint`, a `SliderJoint`, or a `Track`.
